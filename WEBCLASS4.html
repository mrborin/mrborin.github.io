4. Form labels (interacting with users)



4-1
The form is the data that can be entered by the viewer to the server, so that the server-side program can handle the data that the form is passing through.
Grammar:
<form method= "transmission mode" action= "server file" >
Explanation:
1.<form> : <form> tag is presented in pairs, starting with <form >, ending with </form >.
Action: the data entered by the viewer is sent to the site, such as a PHP page (save. PHP).
3. Method: method of data transmission (get/post).

4-2
The <input > input box
Text input box, password entry box
Text input boxes are used when users type letters, Numbers, etc. into the form.The text box
can also be converted into a password entry box.
Grammar:
The <form >
"Input type="text/password "name =" name "value ="text "/ >
</ form >
1, type:
When type= "text", the input box is the text input box;
When type= "password", the input box is the password input box.
2. Name: name the text box for the background program ASP and PHP.
3. Value: set default values for text input boxes.(usually prompt)

4-3
Text fields that support multi-line text input
Text input fields need to be used when a user needs to enter a large text in a form.
Grammar:
<textarea rows= "row number" cols= "column number" > text </textarea>
1. <textarea> tag is presented in pairs, beginning with <textarea >, ending with </textarea >.
2. Cols: the number of columns in a multi-row input field.
3. Rows: the number of rows in a multiline input field.

4-4
Single box, check box
Grammar:
<"Input type="radio/checkbox "value =" value "name =" name "checked ="checked"/>
1, type:
When type= "radio", the control is a single box
When type= "checkbox", the control is the checkbox
2. Value: value to submit data to the server (used by background program PHP)
3. Name: name the control for the background program ASP and PHP

4. Checked: when checked= checked, this option is selected by default
Note: in the same group of radio buttons, the name value must be consistent, so that the single button of the same group can play the single option.

4 - 5
Drop-down list box, save space
Drop-down lists are also used in web pages, which can effectively save web pages.You can choose both.
Syntax: <option value= "submit value" > option </option>
The submit value is the value submitted to the server, and the option is the value displayed.

Use the drop-down list box for multiple choices
The drop-down list can also be multi-selected, setting multiple="multiple" attributes in the <select> label, You can select multiple options by pressing the Ctrl key at the same time to select multiple options.

4 - 7
There are two buttons to use in the form: submit button, reset.
1. Submit button: when the user needs to submit the form information to the server, the submit button needs to be used.
Grammar:
<input type="submit" value= "submit" >
Type: only when the type value is set to submit
Value: the text displayed on the button
2. Reset the button and reset the form information
When the user needs to reset the form information to the initial state, such as when the user enters "user name", it is found that the handwriting is wrong, and the reset button can be used to restore the input box to the initial state.Just set the type to "reset".
Grammar:
<input type="reset" value= "reset" >
Type: the button only has a reset when the type value is reset to reset
Value: the text displayed on the button
